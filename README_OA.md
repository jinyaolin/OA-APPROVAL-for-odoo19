# Odoo 19 OA Approval 項目

## 項目狀態

✅ **已完成重建並驗證** - 系統已準備好用於移植

## 快速開始

### 啟動系統

```bash
# 啟動 Odoo 服務
./start.sh

# 訪問系統
# URL: http://127.0.0.1:8069
# 帳號: admin / admin
```

### 驗證安裝

```bash
# 運行驗證腳本
python verify_oa_approval.py
```

### 移植到其他機器

```bash
# 導出系統
./export_for_porting.sh

# 導出的文件在 oa_approval_export/ 目錄
# 詳見該目錄中的 README_PORTING.txt
```

## 功能列表

### ✅ 已實現功能

1. **動態表單系統**
   - 表單類別管理
   - 表單模板設定
   - 動態字段配置

2. **表單建立流程**
   - 精靈式表單建立
   - 動態下拉選單（根據字段類型顯示相關選項）
   - 草稿階段可編輯

3. **審核工作流**
   - 線性審核流程
   - 特定審核者指定
   - 角色審核（直屬長官/部門經理）
   - 審核通過/駁回功能

4. **權限管理**
   - 三級權限（用戶/經理/管理員）
   - 管理員自動分配 OA 權限
   - 群組基礎存取控制

5. **審核紀錄**
   - 完整審核歷史
   - 審核詳情查看
   - 當前狀態顯示

6. **郵件通知** (佇列處理)
   - 待審核通知
   - 審核通過通知
   - 審核駁回通知
   - 非同步發送，不阻塞 UI

7. **數據完整性**
   - 選項重複問題已修正
   - 資料驗證機制
   - 錯誤處理

## 已知限制

1. **郵件模板功能** - 目前暫時禁用，需要手動配置
2. **審核流程** - 僅支援線性流程，不支援並行審核
3. **多語言** - 主要支援繁體中文
